# Portfolio - Kumar Gaurav

A modern, responsive personal portfolio website built using **React**, **Vite**, and **Tailwind CSS** to showcase projects, skills, and experience.

---

## 🚀 Tech Stack

- **React** – UI library  
- **Vite** – Fast build tool  
- **Tailwind CSS** – Utility-first CSS framework  
- **Framer Motion** – Animations  
- **React Router** – Routing  
- **React Icons** – Icon library  

---

## 📦 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18 or higher)
- npm or yarn

---

### 🔧 Installation

```bash
# Clone the repository
git clone https://github.com/Kumar24Gaurav/Portfolio.git

# Navigate to the project directory
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Create production build
npm run build

# Preview production build
npm run preview
